# Player Guide

## Join

1. Get the invite token or invite URL from the GM.
2. Open the web client.
3. Use the Join form with the invite token, email, display name, and password.
4. Confirm the session switcher shows your player name.

## At The Table

- Use the scene tabs for the current scene.
- Move only your owned token.
- Roll from the dice bar or from actor actions when available.
- Use chat for table-visible messages.
- Check Journal for public handouts and recap entries.

## Expected Limits

- GM-only notes, hidden tokens, private actor data, and pending AI proposals should not appear to player accounts.
- If a token or journal is missing, ask the GM to verify visibility before filing a bug.
