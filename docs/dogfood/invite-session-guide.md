# Invite And Session Guide

## GM Invite Flow

1. Select the dogfood campaign.
2. Enter the player's email in Invites.
3. Choose `Player`, `Observer`, `Assistant GM`, or `GM`.
4. Create the invite and share the token or accept URL out of band.

## Player Accept Flow

1. Paste the invite token into Join.
2. Enter email, display name, and password.
3. Submit Join.
4. Confirm the campaign opens and the session switcher shows the accepted account.

## Session Recovery

- If a browser loses realtime, refresh the page.
- If login state is wrong, use the session switcher for demo seats or accept a fresh invite for a real player account.
- If the API restarted, reload the web page and verify the last exported archive still imports.
