# v0.3 Outside Dogfood Readiness

Date: 2026-05-12

## Objective

Ship OpenTabletop Engine v0.3 for outside dogfood. Beta v0.2 is already accepted; v0.3 focuses on unaffiliated GMs running multi-session games, reporting issues, recovering from failures, and continuing play without project maintainer help.

## Scope

| Area | Status | Evidence |
| --- | --- | --- |
| v0.3 verification docs | In progress | This file, `docs/verification/v0.3-dogfood-progress.md`, and `docs/verification/v0.3-dogfood-acceptance.md`. |
| Outside GM onboarding | Ready for dogfood | `docs/dogfood/first-run-gm-guide.md`, `docs/dogfood/run-your-first-session-checklist.md`, `docs/dogfood/invite-session-guide.md`, `docs/dogfood/player-guide.md`, and `docs/dogfood/troubleshooting.md`. |

| Campaign setup UX | Improved | Sidebar archive import keeps status/errors, campaign switching is explicit, and the content tab exposes preview/apply/rollback/delete for safe content imports. |

| AI/plugin/system dogfood | Beta baseline ready | Existing beta API regression covers AI evals, proposal safety, plugin permissions, and system switching; v0.3 acceptance still needs final proof recorded. |

## Blocked Content

- Roll20 UI, brand, assets, sheets, marketplace data, and proprietary workflows.
- Proprietary D&D, D&D Beyond, or other non-SRD content.
- Scraping or auth bypass for third-party services.

## Acceptance Gate

Do not mark v0.3 accepted until `docs/verification/v0.3-dogfood-acceptance.md` contains proof for install/check, API/web runtime, beta campaign import, GM+3 realtime, three-session runbook, export/import after sessions, AI eval, plugin/system smoke, dogfood report bundle, and content import UX.
